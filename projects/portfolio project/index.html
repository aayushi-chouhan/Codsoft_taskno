Basic Calculator

The Basic Calculator is a simple, yet functional web-based application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. This project was developed as part of my web development portfolio to showcase my skills in HTML, CSS, and JavaScript.

The primary objective of this project was to create an intuitive and user-friendly interface that replicates the functionality of a physical calculator. The calculator is designed with a clean and modern aesthetic, ensuring ease of use and accessibility for all users.

Features:

  • Basic Arithmetic Operations: Addition, Subtraction, Multiplication, Division
  • Responsive Design: Ensures usability across different devices and screen sizes
  • Keyboard Support: Users can perform calculations using both mouse clicks and keyboard inputs
  • Clear Functionality: Easily reset the calculator to perform new calculations

Technologies Used:

  • HTML: Provides the structure of the calculator, including buttons and display areas
  • CSS: Styles the calculator with a modern and clean design, ensuring responsiveness
  • JavaScript: Implements the logic for performing calculations and updating the display in real-time

This project allowed me to deepen my understanding of JavaScript event handling and DOM manipulation. It also provided an opportunity to practice CSS for creating a responsive and visually appealing layout. The Basic Calculator is a testament to my ability to build functional web applications from scratch and my commitment to creating user-centric designs.
